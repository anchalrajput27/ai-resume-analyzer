# AI Resume Analyzer

AI Resume Analyzer is a hackathon project that helps job seekers
analyze their resumes against job descriptions using ATS-style
scoring and keyword matching.

## Features
- Resume upload (PDF/TXT)
- Job description analysis
- ATS score (0â€“100)
- Missing keyword suggestions
- Optimized resume preview
- Public career tips feed

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- Replit

## How It Works
Users upload a resume and paste a job description.
The system compares keywords, calculates an ATS score,
and suggests improvements.
